Accessing Local Servers over LAN (WiFi)

If you start a web server in the terminal and want to access it from your phone or another device on the same WiFi network, you need to expose it to your network by binding it to 0.0.0.0 (or using a specific flag).

  • Vite / React / Vue: npm run dev -- --host
  • Next.js: npm run dev -H 0.0.0.0
  • Python HTTP Server: python -m http.server 8000 --bind 0.0.0.0
  • Flask: flask run --host=0.0.0.0
  • Django: python manage.py runserver 0.0.0.0:8000

Once started, open the Ports tab at the bottom and click the Local LAN button to open it instantly on any device.

Cloudflare Tunnels

Clicking Cloudflare creates a secure, public HTTPS URL. Anyone on the internet can access this URL, bypassing network restrictions completely.
